# How to create a bootable EFI shell

I am going to do this using libvirt, so I need to:
- make an image file
- add a partition to it
- add a file system
- mount it
- create the appropriate directory hieracrchy
- download the efi shell to it


1. Create raw image file
```bash
# dd if=/dev/zero of=just-efi.raw iflag=fullblock bs=1M count=1000
dd if=/dev/zero of=image.img iflag=fullblock bs=1M count=100 && sync
dd if=/dev/zero of=just-efi.raw bs=1M count=1k
```

2. Create a partition
```bash
parted -s just-efi.raw -- mklabel msdos \
     mkpart primary fat32 0% 100% \
     set 1 boot on \
     set 1 esp on
```

3. Create the EFI file system
```bash
mkfs.fat -F 32 just-efi.raw
```

4. Create a mount point for this disk
```bash
mkdir -p /mnt/just-efi
```

5. Mount the disk at /mnt/just-efi
```bash
mount just-efi.raw /mnt/just-efi
```

6. Create a directory structure for a bootable EFI
```bash
mkdir -p /mnt/just-efi/EFI/BOOT
```

7. Change into that directory and download the tianocore shell.efi
```bash
cd /mnt/just-efi/EFI/BOOT
curl -L -O https://raw.githubusercontent.com/tianocore/edk2/UDK2018/ShellBinPkg/UefiShell/X64/Shell.efi
```

8. Unmount the disk
```bash
umount /mnt/just-efi
```

```bash
qemu-img create -f qcow2 /var/lib/libvirt/images/almaefishell.qcow2 100G
```
